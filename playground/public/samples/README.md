# Sample documents

Place test files here to view in the playground:

- **sample.pdf** – normal PDF (or use any PDF URL)
- **sample.docx** – Word document for mammoth preview
- **sample-scanned.pdf** – image-only PDF to test OCR (optional)
- **sample.pptx** – PowerPoint (optional; needs conversion for viewing)

If these files are missing, use "Load from file" in the playground to pick a document from your machine.
